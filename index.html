About

This site is built to make physics and aerospace concepts feel friendly and inclusive. Each problem can include:

  • Plain-language goal (what the question is really asking)
  • Step-by-step explanation (no skipping steps)
  • Why it matters (real life + aerospace connections)

Tip: Click any card to open the full screenshot.

How to add more problems

  1. Put new screenshots into the images/ folder.
  2. Add a new entry in problems.json (copy/paste an existing one).
  3. Commit/push changes (GitHub Pages updates automatically).

If you want, I can also auto-fill titles/topics and write simple explanations for each screenshot.
